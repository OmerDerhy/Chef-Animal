## Merges Chef 👨‍🍳🐾

Welcome to the **Merges Chef** repository! This is a 2D Unity mobile game for iOS and Android where players merge items and interact with cute animal chefs. 

https://apps.apple.com/il/app/merges-chef/id6760717502


## 🎮 About The Game

**Merges Chef** is an engaging puzzle merge game developed in Unity. Players combine various food items and ingredients to help their animal chefs cook up delicious dishes! 

### Key Features
- **Merge Mechanics:** Combine identical items to upgrade them into higher-level ingredients or dishes.
- **Cute Animal Chefs:** Unlock and play with adorable animal characters.
- **Mobile Optimized:** Built specifically for iOS and Android devices.
- **Monetization & Analytics:** Integrated with Unity IAP for in-app purchases and ironSource (LevelPlay) for advertisements.

## 🛠️ Built With

- **Engine:** [Unity](https://unity.com/) (2D Pipeline)
- **Platform:** iOS, Android
- **Ads & Analytics:** Unity Services, ironSource (LevelPlay)
- **Version Control:** Git

## 📂 Project Structure

- `Assets/` - Contains all game assets including scripts, sprites, animations, and prefabs.
- `Packages/` - Unity package dependencies (e.g., Unity 2D Animation, Burst, Collections).
- `ProjectSettings/` - General Unity project configuration files.
- `Merges Chef Android.apk` / `.aab` - Android build files.

## 🚀 Getting Started

### Prerequisites
- Unity Editor (Check `ProjectVersion.txt` for the exact version)
- iOS and Android Build Support modules installed in Unity Hub
- Visual Studio / JetBrains Rider for script editing

### Installation
1. Clone this repository to your local machine.
2. Open Unity Hub and add the project folder.
3. Open the project in the Unity Editor.
4. If there are missing packages, Unity will automatically resolve and download them via the Package Manager.

### Building for Android
1. Go to **File > Build Settings**.
2. Select **Android** and click **Switch Platform** (if not already selected).
3. Connect your Android device or set up an emulator.
4. Click **Build and Run** to deploy the `.apk` directly to your device, or **Build** to generate an `.aab` for the Google Play Store.
